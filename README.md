# Coin Change

Simple coin change algorithm.

Return the change given a coffee price and euro inserted.

## Prerequisites

* Python 3

## Usage
#### Example
```
>>> coin_change(coffee_price=3.14, eur_inserted=10)
{2: 3, 0.5: 1, 0.2: 1, 0.1: 1, 0.05: 1, 0.01: 1}
```
